# Subtask 05 - Fechar documentacao e metadados de release

## Objetivo

Preparar a extensao para distribuicao com documentacao minima e metadados coerentes.

## Arquivos principais

- `package.json`
- `README.md`
- `CHANGELOG.md`
- `LICENSE`
- `media\` (se houver icone)

## Escopo

- revisar categorias, palavras-chave, publisher e metadados de marketplace
- criar um `README.md` com instalacao, uso, comandos e configuracoes
- criar `CHANGELOG.md`
- corrigir placeholders ou dados incompletos de licenca
- adicionar icone se entrar no escopo do release

## Fora de escopo

- criar site externo ou documentacao extensa

## Criterio de pronto

- a extensao fica compreensivel para um usuario novo
- os metadados do `package.json` ficam adequados para empacotamento
- o pacote nao depende de placeholders esquecidos
