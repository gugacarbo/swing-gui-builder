# Subtask 06 - Remocao de componentes e atalhos basicos

## Objetivo

Melhorar a usabilidade do canvas com remocao rapida de componentes e atalhos para operacoes frequentes.

## Arquivos principais

- `webview\main.js`

## Escopo

- permitir remover o componente selecionado com teclado
- evitar conflito entre atalhos globais e inputs do painel de propriedades
- considerar atalhos iniciais como salvar, gerar ou duplicar, se o comportamento for simples e seguro

## Fora de escopo

- sistema completo de keymap customizavel

## Criterio de pronto

- o usuario consegue remover componentes sem recriar o layout
- os atalhos nao quebram a edicao de campos de texto
- o estado salvo e gerado continua consistente apos remocoes
